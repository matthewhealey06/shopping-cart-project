# Shopping Cart

An e-commerce cart that fetches products from an external API and calculates totals dynamically.

## Features
- Fetches products from FakeStore API
- Quantity adjustment with +/- buttons
- Running subtotal, shipping, and total calculations
- Conditional free shipping over £100
- Remove items by reducing quantity to zero
- Event delegation for dynamic elements
- Responsive design

## Built with
- HTML
- CSS
- JavaScript
- FakeStore API
